Overview

Phase I execution is supported by a modern local compute environment and secure cloud-ready workflows.

PromptGenix Phase I focuses on automated analysis pipelines (RNA-Seq + flow cytometry) and a literature-aware Hypothesis Generator. The work is designed to run reproducibly on a dedicated local workstation for rapid iteration, with optional secure deployment in a cloud environment (e.g., AWS VPC) for scale testing.

Local development environment

  • Dedicated workstation: high-memory Mac Studio-class system suitable for omics workflows
  • Core toolchain: Python/R analysis stack; versioned environments; reproducible outputs
  • Pipelines supported: RNA-Seq (QC → alignment → DEG → enrichment) and flow cytometry analysis
  • Report generation: exportable HTML/PDF outputs for reviewers and collaborators
Local-first Reproducible runs HTML/PDF reporting

Secure & scalable deployment option

  • AWS-ready: design supports deployment in a secure VPC for scale and collaboration
  • Data separation: public-data validation in Phase I; sensitive datasets remain external
  • Controlled access: role-based accounts; audit-friendly storage patterns (as applicable)
  • Portability: pipelines and configs can move between local and cloud execution
Phase I policy alignment: validation will use GEO/SRA/FlowRepository public datasets to avoid IP and data-use constraints while demonstrating feasibility and performance.
Resources supporting Phase I

Software, datasets, and operational supports aligned to SBIR deliverables.
