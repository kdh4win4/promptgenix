Use Cases – Public-data Demos (Phase I)

PromptGenix LLC · Contact: dohoon.kim1@icloud.com · promptgenix.org
Why these demos

Phase I demonstrates traceable hypothesis rankings using public datasets and public literature signals.

Each use case follows the same principle: analysis pipelines generate standardized evidence objects, the Bayesian decision engine computes posterior confidence with a credible interval, and the LLM provides evidence-constrained explanation (no evidence → no claim).

What the output looks like

  • Posterior confidence: reported as a probability (or distribution summary) per hypothesis.
  • Uncertainty: 95% credible interval (CrI) for confidence or effect aggregation.
  • Coverage: supporting vs. conflicting evidence counts and weighted strength.
  • Traceability: every ranked item links to evidence objects + citations.
  • Actionability: recommended next-step experiments/analyses and “data needed” if weak.
Posterior 95% CrI Coverage Evidence links

Bayesian engine (core math)

For each hypothesis $H$, PromptGenix uses priors from structured biology and updates with quantitative evidence $E$ from public datasets.

Bayesian update
$$P(H\mid E)=\frac{P(E\mid H)\,P(H)}{P(E\mid H)\,P(H)+P(E\mid \neg H)\,P(\neg H)}$$
Evidence aggregation (weighted)
$$\text{Score}(H)=\sum_{j=1}^{m} w_j \cdot s(E_j),\quad w_j \propto \text{QC}_j \times \text{repro}_j \times \text{context}_j$$
Constraint: LLM explains rankings using approved evidence objects and citations; it does not set $P(H)$ or $P(H\mid E)$.

Use Case 1

Bulk RNA-Seq: disease vs control (public GEO/SRA)

Scenario

A user asks: “Which immune pathways are most likely driving severity in Disease X?” PromptGenix retrieves relevant public bulk RNA-Seq studies and constructs evidence objects from differential expression + pathway signals.

  • Hypothesis template: “Pathway $P$ is upregulated in Disease X and explains severity.”
  • Evidence: pathway enrichment scores, effect sizes, p-values/FDR, cross-study consistency.
  • Output: ranked pathways with posterior confidence + CrI + coverage.
DE & GSEA evidence Cross-study reproducibility Posterior ranking

Likelihood from quantitative evidence
$$\log\frac{P(E\mid H)}{P(E\mid \neg H)}=\alpha\cdot z_{\text{pathway}}+\beta\cdot \text{consistency}+\gamma\cdot \text{QC}$$
Next-step recommendation: propose 1–2 validation experiments (e.g., targeted qPCR panel, cytokine assay, perturbation), tailored to strongest evidence contexts.
Use Case 2

scRNA-Seq: cell-type specific mechanism (public scRNA studies)

Scenario

A user asks: “In Disease Y, which cell type most plausibly drives interferon-like signatures?” PromptGenix aggregates scRNA evidence across studies to evaluate a cell-type mechanism hypothesis.

  • Hypothesis template: “Cell type $C$ shows elevated ISG program in Disease Y.”
  • Evidence: cell-specific pathway scores, marker effect sizes, study-to-study robustness.
  • Output: ranked cell types with posterior confidence + CrI + coverage.

Next-step recommendation: propose targeted sorting + validation panel, or orthogonal assays matched to the identified cell type.
Use Case 3

High-dimensional flow cytometry: phenotype to mechanism (public FlowRepository)

Scenario

A user asks: “Which immune subset expansion is most likely associated with outcome in Cohort Z?” PromptGenix runs standardized cytometry workflows to derive evidence objects and integrates them with literature priors.

  • Hypothesis template: “Subset $S$ is expanded in outcome-positive vs outcome-negative subjects.”
  • Evidence: effect sizes, uncertainty, batch/study normalization stability, cross-cohort signals.
  • Output: ranked subsets with posterior confidence + CrI + coverage.

Phase I evaluation summary

Measurable deliverables tied to these use cases.

Deliverable format: Each demo run produces a reviewer-ready HTML/PDF report including (1) ranked hypotheses, (2) posterior + 95% CrI, (3) supporting vs. conflicting coverage, (4) traceable evidence links and citations, and (5) recommended next steps.
